TDS Pro — Changelog
===================

v3.0.0 (April 2026) — Major Release
─────────────────────────────────────
NEW FEATURES
  + Complete UI redesign — Tally Prime / Zoho Books style
  + Dark sidebar with collapsible mode (56px icon-only)
  + Professional dashboard — 6 KPI cards, bar chart, donut chart
  + Alert banners — overdue, challan pending, certificate expiry
  + PAN autocomplete engine with entity-type detection
  + Dynamic TDS Rules Engine — 38 rules, zero hardcoded rates
  + Section 206AA (no PAN → 20%) and 206AB (no ITR → higher rate)
  + Form 16A HTML certificate generator (Rule 31(1)(b)) + bulk
  + FY 2025-26 surcharge slabs + marginal relief (Section 192)
  + NIL return FVU generator (CBDT mandatory filing)
  + 27Q for NRI payments (Sections 195, 196A-D)
  + Interest Calculator — Section 234A (1.5%/month) + 234E (₹200/day)
  + Excel import with rules-engine rate validation
  + Tally Prime journal format export
  + HMAC-SHA256 license key system — 4 tiers (Trial/Basic/Pro/Ultimate)
  + AES-256 encrypted TRACES credentials (machine-specific key)
  + Daily auto-backup (30-day retention)
  + Toast notifications + loading overlay
  + Global crash handler → crash.log

FIXES
  + Solution Release config entries for all 4 projects
  + AppData-based data path (no write to Program Files)
  + Enter key → Tab navigation on all form fields
